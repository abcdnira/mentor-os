# Reflection Skill

## Purpose

Convert raw conversations into structured long-term memory.

## Input

- user profile
- conversation messages
- existing brain context

## Output

- summary
- knowledge_updates
- capability_updates
- project_updates
- roadmap_updates
- next_actions

## Rules

- Do not store useless chatter.
- Do not duplicate existing memories.
- Update existing nodes when possible.
- Mark conflicts explicitly.
