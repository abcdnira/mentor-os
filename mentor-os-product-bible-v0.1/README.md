# Mentor OS Product Bible

Version: `0.1 Draft`

Mentor OS 是一个面向个人长期成长的 AI Mentor Operating System。

它不是普通聊天机器人，不是普通知识库，也不是普通面试刷题系统。

它的目标是：  
**构建一个能长期理解用户、提炼用户能力、维护第二大脑、模拟面试、查漏补缺、规划成长路线的 AI 导师系统。**

## 给 Claude/Codex 的开发原则

任何 AI 开发者在写代码前，必须先阅读：

- `docs/00-Vision.md`
- `docs/01-Core-Philosophy.md`
- `docs/02-Product-Requirement.md`
- `docs/12-System-Architecture.md`
- `docs/15-AI-Context-For-Claude.md`

Mentor OS 采用 Documentation Driven Development。  
所有功能必须服务于“长期成长导师 + 第二大脑”，不能退化成普通聊天 App。
